About the System

This system is an IoT and machine learning based water quality monitoring platform. It continuously collects real-time sensor data including pH level, total dissolved solids (TDS), temperature, and turbidity from a cloud-connected IoT device via ThingSpeak.

The data is processed using a trained TensorFlow Lite machine learning model that classifies water quality into categories such as Drinking, Irrigation, Washing, or Unsafe.

The dashboard provides real-time monitoring, historical analysis, and AI-based predictions to support decision-making in water safety management systems.

System Features
  • Real-time IoT sensor data visualization
  • Cloud data integration via ThingSpeak API
  • Machine learning-based water classification
  • Historical trend analysis
  • Interactive data charts
How It Works
1. Collect
IoT sensors measure pH, TDS, temperature, and turbidity in real-time, pushing data to ThingSpeak cloud.
2. Analyze
Data is fetched and processed through trend analysis, threshold checks, and historical comparison.
3. Classify
A TensorFlow Lite model classifies water as Drinking, Irrigation, Washing, or Unsafe.
4. Alert
The dashboard surfaces real-time alerts and predictions so you can act immediately on water quality changes.
